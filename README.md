# Startpage : Custom New Tab for any Browser (tested on Chromium and LibreWolf)

Yet another minimal startpage (now with multitheme!):
- Font: SpaceMono Nerd Font
- Colour Scheme(s): Depends on the cover image (Auto Adapt under progress)
